## 一、基本场与符号

* 记张力场（或张力密度场）为 (T(\mathbf{x},t))，定义在空间位置 (\mathbf{x}) 与时间 (t) 上。
* 张力路径的叠加态可表示为一组正交（或非正交）基函数 ({\phi_i(\mathbf{x})}_{i=1}^N)，每条路径有权重 (a_i(t))。

[ T(\mathbf{x},t)=\sum_{i=1}^N a_i(t),\phi_i(\mathbf{x}) + \eta(\mathbf{x},t) ]

其中 (\eta(\mathbf{x},t)) 为小尺度随机扰动或外界噪声项。

* 定义局部复杂度或结构度量（用于阈值判定）：

[ C(t)=\int_{\Omega}\mathcal{F}\big(T(\mathbf{x},t)\big),d\mathbf{x} ]

常用函数 (\mathcal{F}) 的候选：二范数 (\mathcal{F}(T)=|T|^2)、信息熵型函数或加权绝对值。

---

## 二、张力场的变分动力学

假设存在作用泛函（能量型或功能型） (S[T])，张力场沿使 (S) 减小的方向演化，同时接受外界驱动 (F_{ext})：

[ \frac{\partial T}{\partial t}(\mathbf{x},t) = -\gamma,\frac{\delta S}{\delta T(\mathbf{x},t)} + F_{ext}(\mathbf{x},t) + \xi(\mathbf{x},t) ]

其中 (\gamma>0) 为阻尼系数，(\xi) 为噪声。取一个简单的泛函示例：

[ S[T]=\int \bigg( \frac{1}{2}|\nabla T|^2 + V(T) \bigg),d\mathbf{x} ]

则动力学为（类反应-扩散形式）：

[ \frac{\partial T}{\partial t} = \gamma\Delta T - \gamma V'(T) + F_{ext} + \xi ]

这里的势能项 (V(T)) 决定了张力的多稳态特性（例如双稳态用于描述程序的开/关）。

## 三、程序（Program）定义与激活机制

把每种生物/功能程序看作一个二值或连续激活变量 (P_k(t))（(k=1,\dots,M) 表示不同程序）。定义一个投射函数als（或功能算子） (\Phi_k[T]) 表征张力场对第 (k) 个程序的驱动强度：

[ \Phi_k[T]=\int w_k(\mathbf{x}),\mathcal{G}_k\big(T(\mathbf{x},t)\big),d\mathbf{x} ]

其中权重函数 (w_k) 描述程序对不同空间位置的敏感度。

程序激活由阈值关系决定：

[ P_k(t)=\mathcal{S}\big(\Phi_k[T(t)] - \Theta_k\big) ]

(\mathcal{S}) 是平滑化的激活函数（如 Sigmoid），而 (\Theta_k) 为第 (k) 项的激活阈值。

为了表示程序的时间演化，引入动力学：

[ \tau_k\frac{dP_k}{dt} = -P_k + \mathcal{S}\Big(\sum_{j}A_{kj}P_j + \Phi_k[T] - \Theta_k\Big) ]

其中：

* (A_{kj}) 为程序间耦合矩阵（正值表示促进，负值表示抑制）；
* (\tau_k) 为程序响应时常数。

## 四、生长-停止（以体积或质量为例）的简化原型

设生长量为 (G(t))（例如体积），其速率由当前张力驱动项决定：

[ \frac{dG}{dt} = r,G,\big(1-\frac{G}{G_{\max}}\big) \cdot S_g(\Phi_{grow}[T]-\Theta_{grow}) - d,G ]

其中：

* 第一项为 logistic 生长调节，
* (S_g) 为平滑开关（sigmoid），当张力驱动低于阈值时，生长项被抑制。

若定义复杂度 (C(t)) 与 (\Phi_{grow}) 关系为递增，便可建立：当 (C(t)\ge T_c) 时，(S_g\to 0)，导致生长自发停止。

## 五、衰老、疾病与死亡的程序链式激发

将衰老视为一系列程序 (P_{age}^{(1)},P_{age}^{(2)},\dots) 的依次激活。程序链的激发可由耦合矩阵 (A) 与外源干扰共同决定：

[ \frac{d\mathbf{P}}{dt} = -\Lambda\mathbf{P} + \mathcal{S}(A\mathbf{P} + \Phi[T] - \Theta) ]

当局部张力场发生失稳（例如 (V''(T)<0) 的区域），便会增加某些 (\Phi_k[T])，触发错误补偿程序，从而出现疾病动力学。

死亡可以建模为最后一个强抑制程序 (P_{death}) 的激活，其激活会导致关键变量（能量流、修复速率等）迅速衰减到无法逆转的状态。

## 六、外界耦合与远程微扰影响

外界对生物体的影响通过 (F_{ext}(\mathbf{x},t)) 或直接改变势能 (V(T))、阈值 (\Theta_k) 来建模。例如周期性气候信号可表示为：

[ F_{ext}(\mathbf{x},t)=\sum_{m} s_m(\mathbf{x}),\cos(\omega_m t + \phi_m) + f_{stoch}(t) ]

微小且长期累积的外场改变可通过慢变量（参数漂移）作用，使体内复杂度 (C(t)) 缓慢横越临界阈值，从而改变程序激活序列。

## 七、简单示例：单区室、单程序的可视化模型

取单一空间区间 (\Omega)，令 (T(t)) 为区间平均张力，定义：

[ \frac{dT}{dt} = -\gamma V'(T) + F_{ext}(t) + \xi(t) ]
[ P(t)=\frac{1}{1+e^{-\kappa(\Phi[T]-\Theta)}} ][ \frac{dG}{dt}=rG(1-\frac{G}{G_{\max}})\cdot P(t)-dG ]

通过数值仿真可以观察到：随着外场或内部累积（改变 (\Phi[T])），系统出现从增长态到停滞态的相变；加入噪声与周期项可再现生物节律或年际产量波动。

## 八、参数说明与可观测量

* (\gamma)：张力场阻尼/响应速率
* (V(T))：张力势能（决定多稳态）
* (w_k(\mathbf{x}),\Phi_k[T])：程序感受核/投影函数
* (\Theta_k)：程序激活阈值
* (A_{kj})：程序间耦合矩阵（可通过实测协变量估计）
* (\tau_k)：程序响应时常数

可观测量建议：体积/重量（G）、生长速率、基因表达模式（作为 (P_k) 的代理）、环境驱动时间序列（温度、湿度、光照）。
