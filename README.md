## 核心思想

如果叫你来创造一个像今天一样的宇宙，你会怎么做？

方案一，用巨大无比的计算机去每时每刻写入这些变量和参数，去设计原子、粒子、天体、生命…，结果发现，这些变量和参数的量会多到无穷大个宇宙都装不下。

方案二，就是让——最小单元结构方向守恒，然后，让这些最小单元结构相遇只会耦合，但不会折射和反射；剩下的一切活动，让它自己去宇宙里跑，能跑通，路径能闭合变成稳定态，它就活；跑不通，路径不能闭合，不能变成稳定态，它就死。

显然，任何宇宙制造者都会选择方案二；宇宙也不笨，于是它选择了方案二。


宇宙第一性公理（方向守恒公理）

宇宙中，存在最小结构单元，具有初始运动方向并方向守恒；两个以上最小结构单元耦合，其运动方向是其合成方向；当其解耦，
各最小结构单元回归各自的初始运动方向。

除此以外，宇宙没有其他东西；最小结构单元的这个属性，是宇宙的全部内容；也就是说，宇宙的全部内容均可以由宇宙第一性公理推导出。
我们把宇宙的这个属性唯一性，称为宇宙格式或宇宙格式化。
