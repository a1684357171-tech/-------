## 核心思想（最简版本）

宇宙第一性公理（方向守恒公理）

宇宙中，存在最小结构单元，具有初始运动方向并方向守恒；两个以上最小结构单元耦合，其运动方向是其合成方向；当其解耦，
各最小结构单元回归各自的初始运动方向。

除此以外，宇宙没有其他东西；最小结构单元的这个属性，是宇宙的全部内容；也就是说，宇宙的全部内容均可以由宇宙第一性公理推导出。
我们把宇宙的这个属性唯一性，称为宇宙格式或宇宙格式化。
